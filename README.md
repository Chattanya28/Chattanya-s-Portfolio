## 👩‍💻 About The Portfolio

This is my personal portfolio website showcasing my journey as a **Security Engineer** and **Full-Stack Developer**. It highlights my technical skills, hands-on projects, certifications, and professional experience at the intersection of cybersecurity and modern software development.

Built with a focus on **performance**, **clean design**, and **professional storytelling** — the site serves as a central hub for recruiters, collaborators, and the dev community to see what I'm building.

## 🎯 Purpose

- Demonstrate expertise in **.NET 10**, **Java 21**, **Python**, and **cybersecurity frameworks**
- Showcase key projects like **VIA-YOU** (fintech) and **SENTINEL CORE** (AI anomaly detection)
- Provide a clear path to connect for full-time roles, freelance work, or open-source collaboration
- Reflect a **security-first mindset** in both code and content

## 🛠️ Tech Stack (Portfolio Site)

- **Frontend:** HTML5, CSS3, JavaScript (ES6)
- **Deployment:** Netlify (CI/CD via GitHub)
- **Design Principles:** Responsive layout, accessibility, performance optimized

## 📂 Sections Included

1. **Hero** – Role & tagline
2. **About** – Academic background & philosophy
3. **Skills** – Languages, cybersecurity, frameworks, databases, tools, soft skills
4. **Training** – Bootcamps, CTF participation, leadership roles
5. **Projects** – Featured work with tech stacks and dates
6. **Education & Certifications** – Degrees, scores, Oracle/ISC2/Google certs
7. **Contact** – Call to action & social links
